### 签到

本目录保存了一些自己整理的 PT 站点 qiandao 模板，全日志：

1. 基础签到或登录：
    * 对于需要进行的签到的站点，进行签到；
    * 对于不需要进行每日签到的站点，仅访问主页（防止太久没登录 WEB 被封）
2. 获取站点的个人数据，包括：
    * 用户名
    * 上传量
    * 下载量
    * 分享率
    * 签到状态（一般指获得多少魔力）
    * 做种数
    * 做种体积

本目录仅提供 har 文件，需要配合 https://github.com/shuosiw/unraid/blob/master/qiandao.xml 使用：

1. 手动添加模板，上传 har 文件
2. 将本目录作为模板库导入 qiandao 模板中
